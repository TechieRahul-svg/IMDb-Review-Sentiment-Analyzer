# 🎬 IMDb Movie Review Sentiment Analysis

An end-to-end **Sentiment Analysis project** that classifies IMDb movie reviews as **Positive** or **Negative** using Natural Language Processing (NLP) techniques and Machine Learning.

---

## 🚀 Features
- Cleaned and preprocessed raw text data
- Feature extraction using **TF-IDF**
- Built a **Logistic Regression model (~90% accuracy)**
- Compared with Deep Learning models (RNN, LSTM, BiLSTM)
- Developed an interactive **Streamlit web app**
- Real-time sentiment prediction with confidence scores

---

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- NLP (Text Cleaning, TF-IDF)
- Streamlit (for UI)

---

## 📊 Model Performance
- Accuracy: ~90%
- Balanced Precision & Recall
- Evaluated using Confusion Matrix & Classification Report
